# ✨ Minimal Aesthetic Portfolio Website

A stunning, minimal aesthetic portfolio website featuring clean design, smooth animations, and a captivating **horizontal scrolling technical skills showcase**. Built with vanilla HTML, CSS, and JavaScript for optimal performance and elegance.

## 🎨 **Design Philosophy**

**"Professional Dark Aesthetic"** - This portfolio embodies modern design principles with:
- **Full dark mode** with sophisticated color palette
- Clean, spacious layouts with intentional negative space
- **Professional typography** using Poppins, Montserrat, and Roboto
- **Anime-inspired loading animations** with rotating rings and particles
- Purple and cyan accent colors with glowing effects
- Sleek, modern aesthetic with depth and dimension

## ⭐ **Key Features**

### 🚀 **Horizontal Moving Skills Showcase**
- **Infinite horizontal scrolling** of technical skill icons
- **Smooth, continuous movement** (60-second full cycle)
- **Hover to pause** animation for better interaction
- **Seamless loop** with duplicated content
- **Touch-friendly** on mobile devices
- **Consistent icon styling** with professional devicons

### 🎭 **Anime-Style Animations**
- **Spectacular loading screen** with rotating rings, pulsing core, and floating particles
- **Dynamic progress tracking** with percentage counter and status messages
- **Typing effect** in hero section with multiple role titles
- **Scroll-triggered animations** using AOS library
- **Glowing effects** and particle animations
- **Smooth transitions** with dark theme aesthetics

### 📱 **Responsive Design**
- **Mobile-first** approach
- **Fluid typography** that scales beautifully
- **Touch-friendly** navigation and interactions
- **Optimized layouts** for all screen sizes
- **Performance-focused** for mobile devices

### 🛠️ **Technical Excellence**
- **Vanilla JavaScript** - No framework dependencies
- **Semantic HTML5** structure
- **CSS Custom Properties** for easy theming
- **Intersection Observer** for performance
- **Service Worker ready** for PWA capabilities
- **Accessibility compliant** with ARIA labels
